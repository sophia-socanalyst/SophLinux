# Linux Lab Practice

## Overview
This repository contains my hands-on exercises from my cybersecurity training, focusing on Linux file system navigation, basic commands, and system exploration. The purpose is to build foundational skills required for Security Operations Center (SOC) work and general system security.

## Labs Included

### Lab 1 – File System Navigation
- Commands practiced: `pwd`, `ls`, `cd`
- Learned how to navigate directories and understand system structure.

### Lab 2 – Creating and Managing Files/Folders
- Commands practiced: `mkdir`, `touch`, `rm`, `mv`, `cp`
- Learned to create, rename, move, and delete files and folders safely in a lab environment.

### Lab 3 – Viewing and Reading Files
- Commands practiced: `cat`, `less`, `head`, `tail`, `grep`
- Practiced viewing file contents and searching for specific text, skills essential for log analysis in cybersecurity.

## Tools Used
- Linux Terminal (Ubuntu/Kali VM)
- Virtual Machine environment for safe practice

## Learning Outcome
- Gained hands-on experience with Linux commands  
- Developed foundational skills for SOC analyst and cybersecurity tasks  
- Prepared to progress into network scanning, system monitoring, and security labs
